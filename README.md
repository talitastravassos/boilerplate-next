# Boilerplate for Next JS projects.

## Tools
- Typescript
- Eslint
- Prettier
- Jest
- React Testing Library
- Husky
- Styled Components
- Babel

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
